수집하는 정보

두피케어기록은 서버 계정, 광고 식별자, 분석 이벤트를 수집하지 않습니다. 사용자가 촬영한 두피·모발 사진, 입력한 두피/모발 컨디션, 생활 습관 기록, 케어 루틴과 체크 내역, 메모는 앱 기능 제공을 위해 사용자의 기기 내부에만 저장됩니다.

권한 사용

카메라 권한은 사용자가 두피·모발 사진을 직접 촬영할 때만 사용합니다. 촬영한 사진은 기기 내부 앱 저장소에 보관되며 외부로 전송되지 않습니다.

제3자 제공 및 서버 전송

앱은 광고 SDK, 분석 SDK, 클라우드 동기화, 외부 서버 업로드 기능을 포함하지 않습니다. 어떠한 개인 기록도 사용자 기기 밖으로 나가지 않습니다.

보안

민감한 두피·모발 기록 보호를 위해 4자리 PIN 잠금을 제공합니다. 또한 Android 자동 백업과 기기 이전 백업에서 앱 데이터를 제외하도록 설정해 의도치 않은 외부 복제를 막습니다.

데이터 삭제

앱 내 데이터 초기화 기능으로 모든 기록을 언제든 삭제할 수 있으며, 앱을 삭제하면 기기에 저장된 모든 기록과 사진도 함께 제거됩니다.
