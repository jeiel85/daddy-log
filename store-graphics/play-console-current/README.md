# Play Console Current Package

현재 Play Console 업로드에 사용할 자료 묶음입니다.

| 파일 | Play Console 위치 | 상태 |
| --- | --- | --- |
| `icon-512.png` | 스토어 등록정보 → 그래픽 → 앱 아이콘 | 준비 완료 |
| `feature-graphic-1024x500.png` | 스토어 등록정보 → 그래픽 → 기능 그래픽 | 준비 완료 |
| `release-notes.txt` | 새 버전 → 출시 노트 | 준비 완료 (ko/en/ja/zh) |
| `screenshots/` | 스토어 등록정보 → 태블릿 스크린샷 | TB320FC 실기기 캡처 완료 |

릴리즈 AAB와 릴리즈 노트 TXT는 `.\gradlew.bat :app:exportReleaseToDesktop` 실행 시 바탕화면 `Build` 폴더로 함께 복사됩니다.
